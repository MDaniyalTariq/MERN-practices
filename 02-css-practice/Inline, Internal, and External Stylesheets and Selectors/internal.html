Ways To Add CSS

  1. inline
  2. internal
  3. external

HTML Code Example

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Internal CSS</title>
            <style>
                body {
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: blue;
                }
                p {
                    font-size: 16px;
                    line-height: 1.5;
                }
                .highlight {
                    background-color: yellow;
                }
                .important {
                    color: red;
                    font-weight: bold;
                }
            </style>
        </head>
        <body>
            <h1>Ways To Add CSS</h1>
            <ol>
                <li>inline</li>
                <li>internal</li>
                <li>external</li>
            </ol>
            <h2>Internal CSS</h2>
            <p class="highlight">This paragraph is highlighted using internal CSS.</p>
            <p class="important">This paragraph is styled to be important using internal CSS.</p>
        </body>
        </html>

Output

This paragraph is highlighted using internal CSS.

This paragraph is styled to be important using internal CSS.
