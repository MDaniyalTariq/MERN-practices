What is CSS?

Cascading Style Sheet used to work with appearnce related things of website page,used to enhance the appearnce of web page.

What is cascading?

The cascading in CSS refers to the process where styles are applied hierarchically, allowing more specific rules to override general ones. Styles cascade from parent elements to child elements, with regulations from external stylesheets, embedded styles, and inline styles contributing to the final appearance of elements on a webpage.
