External CSS

This paragraph is highlighted using external CSS.

This paragraph is styled to be important using external CSS.

HTML Code Example

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>External CSS</title>
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
            <h2>External CSS</h2>
            <p class="highlight">This paragraph is highlighted using external CSS.</p>
            <p class="important">This paragraph is styled to be important using external CSS.</p>
        </body>
        </html>

Content of style.css

        body {
            font-family: Arial, sans-serif;
        }

        h1 {
            color: blue;
        }

        p {
            font-size: 16px;
            line-height: 1.5;
        }

        .highlight {
            background-color: yellow;
        }

        .important {
            color: red;
            font-weight: bold;
        }
