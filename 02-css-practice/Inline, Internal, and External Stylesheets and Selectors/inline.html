HTML Code Example

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
            <h2>inline CSS</h2>
            <p style="background-color: yellow;">Lorem, ipsum dolor sit amet consectetur adipisicing elit. Enim, voluptas!</p>
            <p style="color: aqua;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Vitae id neque similique doloremque reiciendis ex, fugiat earum doloribus. Adipisci, libero.</p>
        </body>
        </html>
