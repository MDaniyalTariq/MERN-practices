What is a Table?

Arrangement of data in the form of rown and coloumns is called table

Lets understand the table..

  • Horizontal distribution is called row
  • Vertical distribution is called column
  • Intersection of row and column is called cell

Lets create the table..

Available tags to make table..

  1. table(to create the table in html)
  2. thead(to create the header(headings) of the table)
  3. tbody(actual content of the table)
  4. tfoot(summarizing of the table i.e(total epenses))
  5. tr(to create any row in table)
  6. td(to add the data in table row)
  7. th(to create any heading in thead)
